Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag <b>Negrito/Destaque</b>.
Esta tag <b>Negrito/Destaque</b> não tem significado semântico no código.

Já a tag <strong>Negrito/Destaque</strong> visualmente falando é igual a tag <b>Negrito/Destaque</b>.
Porém a tag <strong>Negrito/Destaque</strong> possui um significado semântico onde faz com o que estiver dentro seja algo forte de extrema importância, não apenas visualmente falando.
Segundo Emersom Lourenço:

Ouvi uma vez que é questão de semântica, mesmo que os dois funcionem de forma parecida. Na Web não são somente usuários humanos que estarão consumindo serviços de seus sites, mas também haverá milhares de robôs interpretando seus códigos.
Estes robôs possuem a visão somente de seus códigos, e neste sentido eles avaliam a semântica do que está escrito e não se a palavra está negrita ou não.

Neste sentido, <b> faria somente a questão do negrito, utilizado em certos textos para uma questão estética. Já <strong> traz um significado maior ao texto que está utilizando, de modo que este funcione como marcador de "isto é importante", ou algo do tipo.

Deve-se então ter cuidado ao utilizar <strong>, se for somente para negritar utilizar <b>, se for para enfatizar utilizar <strong>. Espero ter ajudado.

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag <i>Itálico/Enfase</i>.
Da mesma forma que a tag <b>Negrito/Destaque</b> nao tem significado semântico, a tag <i>Itálico/Enfase</i> também não possui significado semântico.

A tag que terá significado semântico para uma parte de texto em itálico é a tag <em>Itálico/Ênfase</em>, quando se da ênfase a um texto.
Para mais informações pode ler a biblioteca do site de desenvolvimento do Mozilla.

Sublinhado ou Inserido

Nesta frase temos um termo em sublinhado usando a tag <u>Sublinhado/Inserido</u>.
Da mesma forma que a tag <b>Negrito/Destaque</b> e que a tag <i>Itálico/Enfase</i>, a tag <u>Sublinhado/Inserido</u> nao possui um significado semântico.

A tag que terá significado semântico para uma parte de texto em sublinhado, é a tag <ins>Itálico/Ênfase</ins>, onde o significado é que o texto foi inserido, de alguma maneira.
Para mais informações pode ler a biblioteca do site de desenvolvimento do Mozilla.

Marcado ou Relevantes

Nesta frase temos um termo marcado usando a tag <mark>Marcado/Relevante</mark>.
Esta tag <mark>Marcado/Relevante</mark> não tem significado semântico no código.

Também é possível alterar a cor do texto <mark>Marcado/Relevante</mark>, usando um atributo dentro da tag, ficando desta maneira "<mark mark style="background-color: lime>Marcado/Relevante</mark>" desta maneira a marcação fica com a cor "lime" (verde florescente), pode ser usado o padrão de cor rgb com os códigos também. Vale lembrar que desta maneira a cor se altera apenas nesta tag em especifico.

Para mudar a cor de todas as tags <mark>Marcado/Relevante</mark> do código pode ser incluído o código abaixo no bloco <head>Cabeçalho</head>. 

                
    <head>
        <style>
            mark {
                background-color: lime;
            }
        </style>
    </head>
Para mais informações pode ler a biblioteca do site de desenvolvimento do Mozilla.

Riscado ou Deletado

Nesta frase temos um termo riscado usando a tag <del>Riscado/Deletado</del>.
Esta tag <del>Riscado/Deletado</del> possui um significado semântico no código, onde parte do texto riscado foi excluído de um documento.

Para mais informações pode ler a biblioteca do site de desenvolvimento do Mozilla.

Texto Pequeno

Nesta frase temos um termo pequeno usando a tag <small>Texto Pequeno</small>.
Esta tag <small>Texto Pequeno</small> possui um significado semântico, sempre que algo direciona ou indica para "ler as letras pequenas de algum contrato" ou até mesmo ler o rodapé de um documento, ele esta se referindo a uma tag <small>Texto Pequeno</small>.

Texto Sobrescrito e Sub-escrito

Nesta frase temos um termo sobrescrito usando a tag <sup>Texto Sobrescrito</sup>.
Um ótimo exemplo é x20+3=0
Nesta frase temos um termo sub-escrito usando a tag <sub>Texto Sub-escrito</sub>.
Um ótimo exemplo é H2O

Abreviações

Para abreviações usasse a tag <abbr title="Texto Abreviado">TA</abbr>.

Citações pequenas e completas

Para citações pequenas usasse a tag <q>Citação pequena</q>. pode combinar e referenciar com o atributo cite <q cite="Referência">Citação pequena</q>

Para citações completas usasse a tag: 

                    
    <blockquote>
        Citação completa.
    </blockquote>
Do mesmo caso que a citação simples, a citação completa também possui o atributo "cite".
